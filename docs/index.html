The signal engine

Eleven strategies, one signal contract

Every strategy implements the same generate_signal / should_exit interface, so the screener can assign any qualifying stock to whichever one fits its current regime.

Trend-following

4 strategies
TREND_EMAADX≥25

50/200 EMA golden/death cross, ADX-filtered. Multi-day/week hold: no EOD timer.

DUAL_EMAADX 20–27

9/20 EMA cross: fills the calendar gap 4–6 weeks before TREND_EMA would fire.

DONCHIAN_BRK1983 Turtles

20-bar high entry / 10-bar low trailing exit: the original Richard Dennis system.

SUPERTRENDATR band

ATR-band trend flip: the most-used indicator in Indian retail algo trading.

Momentum

3 strategies
MOM_CONT+3% / 2× vol

Next-day continuation after a big move, skipped if the gap already priced it in.

MACD_CROSS12/26/9

Gerald Appel's 1970s crossover: fires earlier in a trend than TREND_EMA/SUPERTREND.

HIGH_52WJegadeesh-Titman

New 52-week closing high on volume, anchoring-bias breakout, academically robust.

2 strategies

Overnight reversion above SMA200, buy the close, sell the next open.

For your own use

Run it in three commands

Clone it, install the deps, launch the dashboard. Only yfinance is required to get live market data: no keys needed for paper trading.

1

Clone & install

Grab the repo and install the Python dependencies.

2

Start the dashboard

One Streamlit command serves all eight pages.

3

Watch the loop

Open localhost:8501: the screener, strategies, and paper trades are all there.

# 1) clone & install
git clone https://github.com/chintan-jikkar/KAIROS.git
cd KAIROS
pip install -r requirements.txt

# 2) run the dashboard (paper trading, no keys required)
streamlit run dashboard/app.py --server.port 8501

# 3) open
→ http://localhost:8501

Not financial advice. KAIROS is a rule-based, purely technical trading system: no sentiment or news inputs. Every signal is a model output; backtested performance is not indicative of future results. All trading today is simulated (paper): no live capital is at risk. Market data is sourced from Yahoo Finance on a best-effort basis and may be delayed, incomplete, or revised.
